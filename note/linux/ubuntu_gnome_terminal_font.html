Ubuntu下设置Gnome-Terminal字体

不知道从什么时候开始，Gnome-Terminal 在默认设置里居然找不到 Monaco 字体了，这个当然不能忍

下面是突破这个限制的方法：

  1. 首先，下载到 Monaco 字体文件，然后将其移动到 ~/.fonts 目录（不存在就自己新建）
  2. 然后使用命令 sudo apt-get install gconf-editor 安装配置编辑器
  3. 键入命令 gconf-editor 启动配置编辑器，展开节点 /apps/gnome-terminal/profiles/Default，找到配置项 font
  4. 点击 font 右侧的值填写框，修改为 Monaco 11
  5. 然后重启 Gnome-Terminal，打开默认设置，这时你就可以看到字体已经显示为 Monaco 了

以后就可以直接在 Gnome-Terminal 里修改字体的大小，只是不能变更为其他字体，否则又得用 gconf-editor 才能改回 Monaco 了
